# Automation Scripts

A collection of automation scripts in python created to simplify and automate common DevOps and operational tasks using simple CLI commands.

## What this repository covers
- EC2 lifecycle management using boto3
- Extracting server metrics and monitoring log files
- Managing Objects in an S3 bucket using boto3

## Tech Stack
- Python

## Use Case
These scripts can be used to automate manual steps and schedule them as cron jobs. They can be used to manage AWS resources in case access to the console is limited.
